# **App Name**: BruteFolio

## Core Features:

- Project Showcase: Display project thumbnails in a masonry or grid layout, emphasizing asymmetry and bold typography.
- Project Detail Pages: Each project gets its own details page. The design of the project details pages uses a full-screen layout. These should focus on imagery with brief project descriptions.
- Contact Form: Simple contact form that integrates directly with an email service to avoid needing a database.
- Unconventional Navigation: Navigation should be unconventional. Prefer a fixed sidebar or unusual placement.
- AI Powered Bio: Incorporate a short bio section. Use a generative AI tool to vary the introduction of the portfolio based on keywords that best describe the user of the page.

## Style Guidelines:

- Implement a brutalist layout with raw, unpolished elements and visible grid structures.
- Use very large, bold sans-serif typography for headings and key information.
- Use a high-contrast color scheme. Consider black and white as dominant colors. Use striking backgrounds to accentuate certain elements.
- Accent color: Electric Blue (#7DF9FF) to highlight interactive elements.
- Integrate Material Design icons for a modern and recognizable feel, but in a stark, monochrome style.
- Minimal animations that emphasize transitions and content loading.
